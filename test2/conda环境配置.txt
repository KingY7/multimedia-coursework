================================================================================
                实验二 Conda环境配置（简化版）
================================================================================

第一步：创建环境
--------------------------------------------------------------------------------
conda create -n exp2 python=3.9 -y


第二步：激活环境
--------------------------------------------------------------------------------
conda activate exp2


第三步：安装所有依赖
--------------------------------------------------------------------------------
pip install numpy scipy opencv-python scikit-learn scikit-image matplotlib seaborn pillow tqdm


第四步：验证安装
--------------------------------------------------------------------------------
python -c "import numpy, scipy, cv2, sklearn; print('环境配置成功！')"


第五步：运行实验
--------------------------------------------------------------------------------
python fourier_descriptor.py


================================================================================
完整命令（复制粘贴运行）
================================================================================

conda create -n exp2 python=3.9 -y
conda activate exp2
pip install numpy scipy opencv-python scikit-learn scikit-image matplotlib seaborn pillow tqdm
python -c "import numpy, scipy, cv2, sklearn; print('环境配置成功！')"


================================================================================
后续使用
================================================================================

每次运行实验前：
    conda activate exp2

运行完毕后：
    conda deactivate


================================================================================
删除环境（如需重新配置）
================================================================================

conda deactivate
conda env remove -n exp2
